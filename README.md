# FASHION STORE SALES ANALYSIS DASHBOARD | EXCEL PROJECT

## Project Overview

This project is an interactive Excel Dashboard created to analyze and visualize fashion store sales data for the year 2022.

The dashboard helps in understanding customer purchasing behavior, sales performance, order trends, and channel-wise performance using Excel data analysis techniques.

The complete project was built using Microsoft Excel with the help of Pivot Tables, Pivot Charts, and Slicers.

## Data Cleaning & Transformation

- Cleaned and organized raw sales data
- Checked and corrected data formats
- Removed inconsistencies from the dataset
- Structured data for efficient analysis
- Created Pivot Tables for data summarization

---

## Dashboard Insights

The dashboard provides analysis of:

- Orders vs Sales Trend (Monthly Analysis)
- Sales Distribution by Gender
- Order Status Analysis
- Top 5 States by Sales
- Age Group vs Gender Analysis
- Orders by Sales Channels

---

## Key Insights

- Women customers contributed the highest sales (64%)
- Amazon generated the highest number of orders
- Maharashtra recorded the maximum sales
- Most orders were successfully delivered (~92%)
- Adult customers contributed the largest share of orders
